# 装备与武器属性表

数据来源：`data/items.yaml`。本文档仅列出可装备的武器和防具，不包含消耗品。

说明：
- `攻击`、`防御`、`生命` 为装备基础属性，不包含词条加成。
- `商店价格` 为空的装备通常来自探索、世界 Boss 或虚空献祭等奖励，不在商店直接出售。
- 负数防御表示该武器会降低防御。
