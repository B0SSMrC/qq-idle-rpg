# 游戏指令与预期输出

本文档记录当前 OneBot 框架下游戏机器人已经支持的全部指令，以及每类指令的预期回复格式。

## 触发规则

- 群聊中需要 `@机器人` 后发送指令；私聊中可直接发送指令。
- 指令支持无前缀或 `/` 前缀，例如 `探索` 与 `/探索` 都可识别。
- 玩家身份按 `user_id` 区分，当前世界按 `group_id` 区分；同一 QQ 号在不同群中属于不同世界进度。
- 和角色有关的成功回复通常会带角色名前缀：

```text
「角色名」
具体回复内容
```

## 指令总览

| 指令 | 别名 | 参数 | 说明 |
| --- | --- | --- | --- |
| `注册 <名字>` | `创建 <名字>` | 角色名 | 创建当前世界中的角色 |
| `探索` | `下潜`、`冒险` | 无 | 消耗体力向地牢深处探索 |
| `状态` | `我`、`角色` | 无 | 查看角色属性、装备、金币和最深层数 |
| `背包` | `物品` | 无 | 查看当前背包物品 |
| `装备 <物品>` | 无 | 物品名 | 装备武器或防具 |
| `购买装备 <物品>` | `购买武器`、`购买并装备`、`买装备`、`买武器` | 物品名 | 购买武器或防具并立即装备 |
| `卸下 <物品>` | 无 | 物品名 | 卸下已装备的武器或防具 |
| `使用 <物品>` | 无 | 物品名 | 使用药水、丹药、符箓等消耗品 |
| `使用 <物品> <数量>` | 无 | 物品名、数量 | 批量使用消耗品，支持 `*数量` |
| `使用 <物品A> <物品B>` | 无 | 多个物品名 | 多物品使用，背包没有时自动购买后使用 |
| `商店` | 无 | 无 | 查看可购买商品 |
| `购买 <物品>` | `买 <物品>` | 物品名 | 购买商店商品 |
| `出售装备` | `一键出售`、`清理装备` | 无 | 出售所有未装备的武器和防具 |
| `回满生命` | `回满血`、`补满生命`、`补满血` | 无 | 使用背包治疗药品补满生命 |
| `回满体力` | `补满体力` | 无 | 花金币购买回气丹并补满体力 |
| `前往 <层数>` | `回层 <层数>`、`去 <层数>` | 层数或 `最深` | 移动到已经探索过的层数 |
| `回到 <层数> 并探索` | 无 | 层数或 `最深` | 回到指定已探索层数后立刻探索 |
| `排行榜` | `排名` | 可选 | 查看当前世界等级榜 |
| `排行榜 深度` | `排名 深度` | `深度` 或 `depth` | 查看当前世界深度榜 |
| `帮助` | `菜单`、`?` | 无 | 查看机器人内置指令菜单 |

## 注册

示例：

```text
注册 cxh
创建 Crazy
```

成功输出：

```text
「cxh」
✅ 角色已创建!发「探索」开始冒险吧~
```

常见错误：

```text
用法:注册 角色名(群里/频道里需 @机器人)
角色名需为 1-12 个字符
你在当前世界已经有角色啦
当前世界已有人叫「cxh」,换一个吧
```

## 探索

示例：

```text
探索
下潜
冒险
```

成功输出会根据本次探索事件变化，包括战斗、宝箱、陷阱、叙事文本、掉落、升级、体力耗尽或战败提示：

```text
「cxh」
🗡️ 【cxh】的下潜 (第35层 → 第39层)

第36层 ⚔️ 遭遇石甲守卫，3回合后取胜  +42exp +28金币  ✨ 掉落【碎星短刃】
第37层 📦 发现宝箱  +18金币
第38层 ⚠️ 踩中陷阱
第39层 🚶 墙上的旧刻痕指向更深处。
──────────────
本次合计:+42exp  +46金币  获得1件物品
❤️ HP 86/120   ⚡ 体力 92
📊 升级 +1!  当前 Lv.12
🏆 最深抵达 第39层
```

战斗胜利会从多种句式中随机选择：

```text
第36层 ⚔️ 石甲守卫 → 3回合击败  +42exp +28金币
第36层 ⚔️ 遭遇石甲守卫，3回合后取胜  +42exp +28金币
第36层 ⚔️ 石甲守卫拦路，被你用3回合解决  +42exp +28金币
第36层 ⚔️ 与石甲守卫短兵相接，3回合胜出  +42exp +28金币
第36层 ⚔️ 斩退石甲守卫，战斗持续3回合  +42exp +28金币
```

可能追加的结尾：

```text
💀 重伤休整,当前层数保留(金币略有损失)
💤 体力耗尽,攒一攒再来下潜~
```

常见错误：

```text
你在当前世界还没有角色,先发「注册 角色名」吧~
```

## 使用

示例：

```text
使用 小治疗药水
使用 攻击符箓
使用 金疮药 3
使用 金疮药 *3
使用金疮药 3
使用 虎骨酒 蛮牛散
使用 金疮药 *3 虎骨酒
```

成功后返回使用结算和最新角色状态面板。治疗类物品会恢复 HP，攻击/防御类物品会显示 Buff，体力类物品会恢复体力。

批量规则：

- 数量必须大于 0。
- 如果背包数量不足，会自动从商店购买缺少的数量后使用。
- 如果购买失败或物品不可用，会记录该物品失败原因，并继续处理后续物品。
- 攻击/防御 Buff 不叠加数值，多次使用仍按最后一次效果覆盖。

预期输出：

```text
「cxh」
✅ 使用结算
・虎骨酒 ×1 (自动购买1个,花费70金币)
・蛮牛散 ×1
・不存在的药 未使用:没有这件物品: 不存在的药
🛡️ cxh  Lv.12  (当前世界)
...
```

常见错误：

```text
用法:使用 物品名
数量必须大于 0
没有这件物品: 小治疗药水
物品名 不是消耗品
背包里没有 物品名
```

## 出售装备

示例：

```text
出售装备
一键出售
清理装备
```

有可出售装备时：

```text
「cxh」
💰 一键出售结算
售出 3 件装备，获得 640 金币
──────────────
· 铁剑 x2  96/件 = 192
· 皮甲 x1  80/件 = 80
当前金币: 1900
```

没有可出售装备时：

```text
「cxh」
💰 一键出售结算
没有可出售的未装备武器/防具。
当前金币: 1260
```

规则说明：

- 只出售未装备的武器和防具。
- 已装备物品不会出售。
- 药水、符箓等消耗品不会出售。

常见错误：

```text
你在当前世界还没有角色,先发「注册 角色名」吧~
```

## 回满生命

示例：

```text
回满生命
回满血
补满生命
补满血
```

该指令只使用背包里已有的治疗类消耗品，不会自动买药。系统会按治疗性价比选择药品，直到生命回满或背包药品用完。

成功输出：

```text
「cxh」
✅ 已回复生命:10 → 120/120
使用:金疮药×1、续命丹×1
🛡️ cxh  Lv.12  (当前世界)
...
```

药品不足时：

```text
「cxh」
✅ 已回复生命:10 → 50/120
使用:金疮药×1
背包回复药已用完,生命尚未回满。
🛡️ cxh  Lv.12  (当前世界)
...
```

生命已满或没有药品时：

```text
生命已满,无需使用药品。
背包里没有可回复生命的消耗品。
```

## 回满体力

示例：

```text
回满体力
补满体力
```

机器人会按商店中 `回气丹` 的单价和体力回复量，自动计算需要购买几颗，并直接把体力补到上限。该指令不消耗背包里的回气丹。

成功输出：

```text
「cxh」
✅ 已回满体力,消耗160金币。
🛡️ cxh  Lv.12  (当前世界)
经验 380
❤️ HP 86/120   ⚡ 体力 100/100
⚔️ 攻击 58   🛡️ 防御 24   💪 战力 82
💰 金币 1100   🏆 最深 第60层
🎒 装备:碎星短刃、巡林轻甲
```

体力已经满时：

```text
「cxh」
体力已满,无需购买回气丹。
🛡️ cxh  Lv.12  (当前世界)
...
```

如果短时间内连续大量使用该功能，会触发爆气负面 Buff：

```text
「cxh」
✅ 已回满体力,消耗160金币。
💥 你触发了「爆气」负面buff:攻击下降15%,防御下降20%,持续10分钟。
🛡️ cxh  Lv.12  (当前世界)
...
💥 爆气:攻击-15% 防御-20% (剩余约10分钟)
```

常见错误：

```text
金币不足(需 160,当前 80)
你在当前世界还没有角色,先发「注册 角色名」吧~
```

## 前往

示例：

```text
前往 35
回层 20
去 最深
前往 最深层
前往 max
前往 deepest
```

成功后会更新当前探索起点，并返回角色状态面板：

```text
「cxh」
已前往第 35 层。
🛡️ cxh  Lv.12  (当前世界)
经验 380
❤️ HP 86/120   ⚡ 体力 92/100
⚔️ 攻击 58   🛡️ 防御 24   💪 战力 82
💰 金币 1260   🏆 最深 第60层
🎒 装备:碎星短刃、巡林轻甲
```

常见错误：

```text
用法: 前往 层数，例如「前往 35」或「前往 最深」
层数必须大于等于 1
你最深只到过第 60 层，暂时不能前往第 80 层
你在当前世界还没有角色,先发「注册 角色名」吧~
```

```text

❤️ HP 86/120   ⚡ 体力 92
🏆 最深抵达 第60层
```

常见错误：

```text
用法: 回到 层数 并探索，例如「回到 35 并探索」
层数必须大于等于 1
你最深只到过第 60 层，暂时不能前往第 80 层
你在当前世界还没有角色,先发「注册 角色名」吧~
```

## 未知指令

当消息 `@机器人` 且没有匹配到任何已注册指令时，机器人会从话术池中随机选择一条回复。

当前话术池：

```text
没听懂这个指令。发送「帮助」查看可用指令吧~
这条咒语还没收录进地牢手册里，发「帮助」看看菜单。
指令好像走岔路了。试试「探索」「状态」「背包」或「帮助」。
```
