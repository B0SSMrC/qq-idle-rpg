# 游戏指令与预期输出

本文档记录当前 OneBot 框架下游戏机器人已经支持的全部指令，以及每类指令的预期回复格式。

## 触发规则

- 群聊中需要 `@机器人` 后发送指令；私聊中可直接发送指令。
- 指令支持无前缀或 `/` 前缀，例如 `探索` 与 `/探索` 都可识别。
- 系统会先识别标准指令；标准指令没命中时，会尝试本地模糊解析常见自然写法，例如 `查看背包`、`重铸武器10`、`购买装备铁剑`、`回到35并探索`。
- 模糊解析只判断“要执行哪个指令”，不会模糊修改物品名、角色名或层数；低置信度输入会继续走未知指令话术，避免误花金币或误出售装备。
- 玩家身份按 `user_id` 区分，当前世界按 `group_id` 区分；同一 QQ 号在不同群中属于不同世界进度。
- 和角色有关的成功回复通常会带角色名前缀：

```text
「角色名」
具体回复内容
```

## 指令总览

| 指令 | 别名 | 参数 | 说明 |
| --- | --- | --- | --- |
| `注册 <名字>` | `创建 <名字>` | 角色名 | 创建当前世界中的角色 |
| `探索` | `下潜`、`冒险` | 无 | 消耗体力向地牢深处探索 |
| `状态` | `我`、`角色` | 无 | 查看角色属性、装备、金币和最深层数 |
| `背包` | `物品` | 无 | 查看当前背包物品 |
| `装备 <物品>` | 无 | 物品名 | 装备武器或防具 |
| `购买装备 <物品>` | `购买武器`、`购买并装备`、`买装备`、`买武器` | 物品名 | 购买武器或防具并立即装备 |
| `卸下 <物品>` | 无 | 物品名 | 卸下已装备的武器或防具 |
| `使用 <物品>` | 无 | 物品名 | 使用药水、丹药、符箓等消耗品 |
| `使用 <物品> <数量>` | 无 | 物品名、数量 | 批量使用消耗品，支持 `*数量` |
| `使用 <物品A> <物品B>` | 无 | 多个物品名 | 多物品使用，背包没有时自动购买后使用 |
| `商店` | 无 | 无 | 查看可购买商品 |
| `购买 <物品>` | `买 <物品>` | 物品名 | 购买商店商品 |
| `出售装备` | `一键出售`、`清理装备` | 无 | 出售所有未装备的武器和防具 |
| `分解装备` | `一键分解装备`、`分解武器`、`分解防具` | 可选范围 | 分解未装备武器或防具，获得装备养成材料 |
| `强化 武器/装备 [次数]` | 无 | 装备槽位、可选次数 | 消耗金币和材料强化当前已装备武器或防具 |
| `升星 武器/装备` | 无 | 装备槽位 | 消耗金币和同名装备或材料提升当前已装备武器或防具星级 |
| `虚空献祭 [1/10]` | `献祭`、`十连献祭`、`献祭十连` | 可选，`1` 或 `10` | 花金币献祭抽取装备 |
| `世界boss` | `世界boss状态` | 无 | 查看当前群所有可挑战世界 Boss |
| `进攻世界boss [编号/名称]` | `攻击世界boss`、`挑战世界boss` | 可选 Boss 编号或名称 | 消耗 50 体力进攻指定世界 Boss |
| `世界boss排行` | 无 | 无 | 查看当前世界 Boss 伤害排行 |
| `重铸 武器/装备 [次数]` | 无 | 装备槽位、可选次数 | 花金币重随机当前已装备武器或防具的词条 |
| `回满生命` | `回满血`、`补满生命`、`补满血` | 无 | 使用背包治疗药品补满生命 |
| `回满体力` | `补满体力` | 无 | 花金币购买回气丹并补满体力 |
| `前往 <层数>` | `回层 <层数>`、`去 <层数>` | 层数或 `最深` | 移动到已经探索过的层数 |
| `回到 <层数> 并探索` | 无 | 层数或 `最深` | 回到指定已探索层数后立刻探索 |
| `排行榜` | `排名` | 可选 | 查看当前世界等级榜 |
| `排行榜 深度` | `排名 深度` | `深度` 或 `depth` | 查看当前世界深度榜 |
| `帮助` | `菜单`、`?` | 无 | 查看机器人内置指令菜单 |

## 模糊指令示例

以下写法会被解析到对应的标准指令：

| 用户输入 | 等价指令 |
| --- | --- |
| `注册角色小明` | `注册 小明` |
| `继续探索` | `探索` |
| `看看状态`、`查看属性` | `状态` |
| `查看背包`、`看背包` | `背包` |
| `穿戴铁剑` | `装备 铁剑` |
| `取下铁剑` | `卸下 铁剑` |
| `买铁剑` | `购买 铁剑` |
| `买装备铁剑`、`购买装备铁剑` | `购买装备 铁剑` |
| `用金疮药3`、`使用金疮药3` | `使用 金疮药 3` |
| `补血` | `回满生命` |
| `看商店`、`打开商店` | `商店` |
| `重铸武器`、`重铸武器10` | `重铸 武器`、`重铸 武器 10` |
| `分解武器`、`一键分解装备` | `分解武器`、`分解装备` |
| `强化武器10` | `强化 武器 10` |
| `升星装备` | `升星 装备` |
| `虚空献祭10`、`十连献祭` | `虚空献祭 10` |
| `去35层`、`回35层` | `前往 35` |
| `回到35并探索` | `回到 35 并探索` |
| `深度排行`、`排行榜深度` | `排行榜 深度` |
| `指令菜单`、`帮助菜单` | `帮助` |
| `boss状态`、`世界boss` | `世界boss` |
| `打世界boss`、`进攻boss`、`进攻世界boss 2` | `进攻世界boss`、`进攻世界boss 2` |
| `boss排行` | `世界boss排行` |

安全限制：
- `出售装备`、`一键出售`、`清理装备` 必须是明确短语，不会因为普通消息里出现“装备”就触发。
- `强化`、`升星` 必须以明确动作词开头，并且参数必须是 `武器`、`装备` 或 `防具`，例如 `武器强化好吗` 不会执行。
- `购买装备` 和 `购买` 会按最长指令优先解析，避免把 `购买装备铁剑` 误当作购买名为“装备铁剑”的物品。
- `回满体力` 必须明确输入，不会因为普通“体力”聊天触发。
- `买不买铁剑`、`去不去35层`、`重铸一下武器` 这类疑问或不够明确的表达不会执行指令。

```text

```

```text
「cxh」
```

```text

```

## 探索

示例：

```text
探索
下潜
冒险
```

成功输出会根据本次探索事件变化，包括战斗、宝箱、陷阱、叙事文本、掉落、升级、体力耗尽或战败提示：

```text
「cxh」
🗡️ 【cxh】的下潜 (第35层 → 第39层)

第36层 ⚔️ 遭遇石甲守卫，3回合后取胜  +42exp +28金币  ✨ 掉落【碎星短刃】
第37层 📦 发现宝箱  +18金币
第38层 ⚠️ 踩中陷阱
第39层 🚶 墙上的旧刻痕指向更深处。
──────────────
本次合计:+42exp  +46金币  获得1件物品
❤️ HP 86/120   ⚡ 体力 92
📊 升级 +1!  当前 Lv.12
🏆 最深抵达 第39层
```

战斗胜利会从多种句式中随机选择：

```text
第36层 ⚔️ 石甲守卫 → 3回合击败  +42exp +28金币
第36层 ⚔️ 遭遇石甲守卫，3回合后取胜  +42exp +28金币
第36层 ⚔️ 石甲守卫拦路，被你用3回合解决  +42exp +28金币
第36层 ⚔️ 与石甲守卫短兵相接，3回合胜出  +42exp +28金币
第36层 ⚔️ 斩退石甲守卫，战斗持续3回合  +42exp +28金币
```

第 80 层及以后每一层都会触发战斗，不会出现只靠宝箱、叙事或其他非战斗事件直接推进层数的情况。

可能追加的结尾：

```text
💀 重伤休整,当前层数保留(金币略有损失)
💤 体力耗尽,攒一攒再来下潜~
```

常见错误：

```text
你在当前世界还没有角色,先发「注册 角色名」吧~
```

## 词条与重铸

新掉落或从商店购买的武器、防具会自动生成 1 条词条。武器使用武器词条池，防具使用防具词条池；词条可能是正面、负面，或同时带有正负效果。正面增益最高约 +20%，负面效果最低约 -15%。

武器词条包含 `嗜血`：玩家每次攻击命中怪物后，会按本次伤害的一定比例恢复生命，恢复值不会超过生命上限。

状态和背包会在装备名后显示词条：

```text
🎒 装备:碎星短刃[嗜血(吸血+12%)]、巡林轻甲[坚壁(防御+18%)]
```

示例：

```text
重铸 武器
重铸 装备
重铸 武器 10
```

每次重铸消耗 200 金币，并替换当前已装备武器或防具的词条。多次重铸只保留最后一次结果；如果金币不足以完成输入次数，会按当前金币能支付的次数执行。

成功输出：

```text
「cxh」
🔨 重铸完成:10次,花费2000金币
旧词条:嗜血(吸血+12%)
新词条:狂战(攻击+18% 防御-8%)
🛡️ cxh  Lv.12  (当前世界)
...
🎒 装备:碎星短刃[狂战(攻击+18% 防御-8%)]、巡林轻甲[坚壁(防御+18%)]
```

常见错误：

```text
用法:重铸 武器/装备 [次数]
次数必须大于 0
槽位只能是武器或装备
金币不足(需 200,当前 50)
当前没有装备可重铸
你在当前世界还没有角色,先发「注册 角色名」吧~
```

## 使用

示例：

```text
使用 小治疗药水
使用 攻击符箓
使用 金疮药 3
使用 金疮药 *3
使用金疮药 3
使用 虎骨酒 蛮牛散
使用 金疮药 *3 虎骨酒
```

成功后返回使用结算和最新角色状态面板。治疗类物品会恢复 HP，攻击/防御类物品会显示 Buff，体力类物品会恢复体力。

批量规则：

- 数量必须大于 0。
- 如果背包数量不足，会自动从商店购买缺少的数量后使用。
- 如果购买失败或物品不可用，会记录该物品失败原因，并继续处理后续物品。
- 攻击/防御 Buff 不叠加数值，多次使用仍按最后一次效果覆盖。

预期输出：

```text
「cxh」
✅ 使用结算
・虎骨酒 ×1 (自动购买1个,花费70金币)
・蛮牛散 ×1
・不存在的药 未使用:没有这件物品: 不存在的药
🛡️ cxh  Lv.12  (当前世界)
...
```

常见错误：

```text
用法:使用 物品名
数量必须大于 0
没有这件物品: 小治疗药水
物品名 不是消耗品
背包里没有 物品名
```

## 出售装备

示例：

```text
出售装备
一键出售
清理装备
```

有可出售装备时：

```text
「cxh」
💰 一键出售结算
售出 3 件装备，获得 640 金币
──────────────
· 铁剑 x2  96/件 = 192
· 皮甲 x1  80/件 = 80
```

没有可出售装备时：

```text
「cxh」
💰 一键出售结算

当前金币: 1260
```

规则说明：

- 只出售未装备的武器和防具。
- 已装备物品不会出售。
- 药水、符箓等消耗品不会出售。

常见错误：

```text
你在当前世界还没有角色,先发「注册 角色名」吧~
```

## 装备养成

### 分解装备

示例：

```text
分解装备
一键分解装备
分解武器
分解防具
```

预期输出：

```text
「cxh」
🧩 分解完成
分解 2 件装备:
· 铁剑 x1 -> 精铁 x1
· 皮甲 x1 -> 精铁 x1

获得材料:精铁 x2
```

规则说明：

- 只分解未装备的武器和防具。
- `分解武器` 只处理武器，`分解防具` 只处理防具，`分解装备` 处理两类。
- 装备稀有度越高，分解获得的材料等级越高。

常见错误：

```text
没有可分解的未装备武器或防具
```


## 世界 Boss

示例：

```text
世界boss
世界boss状态
世界boss排行
进攻世界boss
进攻世界boss 2
进攻世界boss 焚天战魁
攻击世界boss

```

规则说明：
- 每个群可以同时存在多只世界 Boss。
- Boss 属性来自 `data/world_bosses.yaml`，可以通过配置新增、删除、禁用和调数值。
- `进攻世界boss` 默认攻击配置中 `tier` 最低的 Boss。
- `进攻世界boss 2` 会攻击 `tier: 2` 的 Boss，也可以用 Boss 名称选择。
- Boss 被击败后按该 Boss 配置的冷却时间进入冷却。
- Boss 存在时，机器人每 10 分钟合并公告一次所有 Boss 血量和玩家伤害贡献。
- 每次进攻消耗 50 体力。
- 单次进攻会自动战斗到玩家死亡或 Boss 被击败。
- 玩家战败会损失当前金币的 5%，并回满生命。
- Boss 被击败后按玩家累计有效伤害百分比发放奖励。
- 有效贡献达到 1% 后会触发最低金币奖励；如果玩家金币低于当前成长阶段的建议储备，系统会额外提高金币奖励。
- 奖励包括金币、消耗品、武器和防具；世界 Boss 稀有装备掉率高于普通探索，且伤害占比越高掉率越高。
- 高阶 Boss 可通过 `reward_multiplier` 提升金币奖励，通过 `min_gold` 设置有效贡献最低金币，通过 `material_bias` 调整强化材料掉落倾向。

状态输出：

```text
🌑 世界Boss列表
1. 万劫魔君  HP 6150/15000 (41.0%)
2. 焚天战魁  HP 28000/28000 (100.0%)
3. 沧溟龙相  HP 52000/52000 (100.0%)
4. 太虚星君  HP 90000/90000 (100.0%)

发送「进攻世界boss 1」或「进攻世界boss 2」选择目标。
```

进攻后战败输出：

```text
「cxh」
⚔️ 你向世界Boss「万劫魔君」发起进攻,鏖战37回合后倒下。
本次造成 1450 伤害。
💰 损失金币 5%: -420
❤️ 已回满生命。
⚡ 消耗体力 50
Boss剩余 HP: 6150/15000
```

击败 Boss 后会追加奖励结算：

```text
🌑 世界Boss 万劫魔君 已被击败!
🏆 奖励结算
・cxh 56.0% 获得 17640金币、九转还魂丹×2、掉落:血海长刀
```

常见错误：

```text
你在当前世界还没有角色,先发「注册 角色名」吧~
体力不足(需 50,当前 20)
世界Boss已经被击败,等待下一次刷新吧。
世界Boss战斗结算繁忙,请稍后再试。
```

后台配置文件：`data/world_bosses.yaml`

```yaml
- key: burning_warlord
  name: 焚天战魁
  enabled: true
  tier: 2
  title: 进阶
  atk: 240
  def: 100
  base_hp: 18000
  hp_per_active_player: 5000
  cooldown_hours: 8
  active_hours: 48
  reward_multiplier: 2.5
  min_gold: 5000
  material_bias:
    refined_iron: 2
    black_iron: 5
    star_meteorite: 2
    divine_forge_crystal: 0
```

配置字段说明：

- `key`：Boss 唯一标识，创建后不要随意改名，否则会被视为新 Boss。
- `name`：玩家看到的 Boss 名称，也可用于进攻指令。
- `enabled`：是否启用。设为 `false` 后不会刷新新 Boss，也不能被选择进攻。
- `tier`：玩家选择编号和排序字段，必须唯一。
- `title`：难度标签。
- `atk` / `def`：Boss 攻击和防御。
- `base_hp`：基础生命。
- `hp_per_active_player`：按活跃玩家数额外增加的生命。
- `cooldown_hours`：被击败或逃离后的冷却小时数。
- `active_hours`：刷新后存在多久。
- `reward_multiplier`：金币奖励倍率。
- `min_gold`：玩家有效贡献达到 1% 后的最低金币奖励。
- `material_bias`：材料奖励倾向，只支持 `refined_iron`、`black_iron`、`star_meteorite`、`divine_forge_crystal`，数值越高越容易出现。
